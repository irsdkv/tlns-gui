# Install

```bash
$ # git clone git@github.com:irsdkv/tlns-gui.git
$ git clone https://github.com/irsdkv/tlns-gui.git
$ cd tlns-gui
$ python3 -m venv .venv
$ source .venv/bin/activate
$ pip install -e .
$ deactivate
```

# Run
```bash
$ cd tlns-gui
$ source  .venv/bin/activate
$ tlns_gui.py
$ deactivate
```
